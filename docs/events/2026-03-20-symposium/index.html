趣旨

国立国語研究所・次世代言語科学研究センターは、言語学と自然言語処理の協働を目的のひとつに発足しました。メンバーの間では分野を越えた相互理解と学際的研究のため、愉しい議論が日々おこなわれています。今回の研究会で、この輪をさらに広げたいと思っています。

研究会のテーマは「言語モデルの時代の言語の学を考える」です。言語モデルの台頭により、言語学・言語理論の妥当性について賛否様々な主張が沸き起こっています。このテーマはもちろん簡単に答えが出る類のものではありません。言語学者と自然言語処理の研究者が集まって議論することで、前提としている考え方の違いを明らかにし、隠れている共通点を見つけ出し、言語学の未来を考える機会となればと思っています。

アンチハラスメントポリシー

参加者の皆さんには、事前にアンチハラスメントポリシーの確認と同意をお願いいたします。
建設的で刺激的な学際的議論を実現するためには、様々な専門性を持った人が集まり、お互いの分野の価値観やディシプリンに経緯を払い、そして学ぶ必要があります。他分野に対するネガティブな価値判断をおこなわないなど、学術的に誠実な心構えでのご参加をお願いします。
