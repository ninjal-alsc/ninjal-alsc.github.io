2025年度第4回ハンズオンセミナー

September 20, 2025

『Universal Dependencies』に関するハンズオンセミナー

  • 開催期日: 2025年9月20日 (土) 10:00〜12:00
  • 開催場所: 国立国語研究所 多目的室 (東京都立川市緑町10-2) 交通案内
  • 主催: 国立国語研究所 次世代言語科学研究センター・共同利用推進センター・基幹型プロジェクト「実証的な理論・対照言語学の推進」
  • 参加費: 無料
  • 参加資格:特にありません
  • 定員: 定員20名
  • タイトル: 「ChaKi.NET liteを使ったUD Treebank検索チュートリアル」
  • 講師: 伊藤 薫先生（九州大学）
  • 事前準備: 各自　Windows がインストールされたノートPC　をお持ちください。
  • お問い合わせ: 浅原正幸 (国立国語研究所) masayu-a[at]ninjal.ac.jp ( [at] を@に変えてください。)
  • キーワード: 講習会、Universal Dependencies
  • 内容: 『Universal Dependencies』に関するハンズオンセミナーを開催いたします。検索ツール「ChaKi.NET Lite」の機能について紹介します。また、同日午後に Universal Dependencis 研究会を開催します。pdf
  • 遠方から参加される学生さんには旅費の支援の枠を準備しております。masayu-a -at- ninjal.ac.jp （担当：浅原）までお問い合わせください。
